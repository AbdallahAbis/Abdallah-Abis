## ðŸ›  Installation & Set Up

1. Install the Gatsby CLI

   ```sh
   npm install -g gatsby-cli
   ```

2. Fork and/or clone this repo.

   ```sh
   git clone "github repo"
   ```

3. Install dependencies

   ```sh
   yarn
   ```

4. Start the development server

 ###### On Localhost

     ```sh
     gatsby develop
     ```
 ###### On Your Local Network

     ```sh
     yarn dev-local
     ```
